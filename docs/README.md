### Frontend
- `src/app/` - Next.js App Router pages
- `src/components/` - Reusable React components
- `src/hooks/` - Custom React hooks

### Backend
- `src/app/api/` - API routes
- `src/services/` - Service layer for API calls
- `prisma/schema.prisma` - Database schema
